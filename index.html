Welcome

The SCAR-20 is a semi-automatic sniper which is found in Counter Strike: Global Offensive. I managed to create a version for Trouble In Terrorist Town which is a gamemode for Garrys Mod. The stats are as follows:

SWEP.Primary.Recoil = 1.587

SWEP.Primary.Automatic = false

SWEP.Primary.Damage = 31

SWEP.Primary.Cone = 0.003

SWEP.Primary.Ammo = "357"

SWEP.Primary.ClipSize = 20

SWEP.Primary.ClipMax = 40

SWEP.Primary.DefaultClip = 40

SWEP.Primary.Sound = Sound( "scar20_unsil-1.wav" )

SWEP.HeadshotMultiplier = 4

Downloading

Downloading is as simple as 1, 2, 3. All you do is download the file, and put the files in the corresponding folders.
